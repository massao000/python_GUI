# Python GUI 作成中

GUIのライブラリを使うときはPythonで仮想環境を作ることを推奨します

pip を使ったモジュールのダウンロード方法

```Kivy
$ pip install kivy japanize_kivy
```

japanize_kivyはkivyで日本語を使えるようにするもの

```PySimpleGUI
$ pip install pysimplegui
```

```PySide2
$ pip install PySide2
```
